LINE 相關

整合步驟：
  1. 前往 LINE Developers Console，點擊「Create a LINE Official Account」建立官方帳號
  2. 在官方帳號管理後台 → 設定 → Messaging API → 啟用 Messaging API
  3. 回到 LINE Developers Console，進入 Channel，複製 Channel ID、Channel Secret、Channel Access Token
  4. 登入店家後台 → LINE 設定 → 填入上述資訊
  5. 設定 Webhook URL 到 LINE Developers Console
  6. 關閉 LINE Official Account Manager 的自動回應功能
  7. 完成！顧客即可透過 LINE 預約

顧客預約流程非常簡單：
  1. 加入您的 LINE 官方帳號
  2. 點選選單中的「開始預約」
  3. 選擇想要的服務
  4. 選擇預約日期
  5. 選擇指定員工（或不指定）
  6. 選擇時段
  7. 確認預約資訊
  8. 完成！預約前會自動收到 LINE 提醒

請依序檢查以下項目：
  1. Webhook URL：確認 LINE Developers 設定的 URL 正確
  2. Use webhook：確認為 ON（開啟）
  3. 自動回應：必須關閉 LINE Official Account Manager 的自動回應功能
  4. Token 有效：確認 Channel Access Token 未過期
  5. 連線測試：在店家後台的 LINE 設定點擊「連線測試」

功能相關

在店家後台的「員工管理」頁面：
  • 排班設定：設定每位員工的每週上班時間
  • 休息時段：設定午休或其他休息時間
  • 請假管理：設定特定日期的請假
系統會自動根據排班狀況顯示可預約時段，避免重複預約。

訂閱「票券系統」功能後：
  1. 在後台「票券管理」建立票券（折扣券、現金券、免費服務券）
  2. 設定發放數量、使用期限、每人限領數量
  3. 顧客在 LINE 點選「領取票券」即可領取
  4. 消費時，顧客出示「我的票券」中的代碼
  5. 店家在後台進行核銷

訂閱「集點系統」功能後：
  1. 在「店家設定」設定集點比例（例如消費 NT$10 得 1 點）
  2. 顧客完成消費後，點數會自動累積
  3. 顧客可在 LINE 的「會員資訊」查看點數餘額
  4. 店家可在後台手動調整顧客點數

訂閱「自動預約提醒」功能後：
  • 系統會在預約前一定時間自動發送提醒
  • 支援 LINE 訊息和 SMS 簡訊兩種方式
  • 可在「店家設定」調整提醒時間
  • 有效降低顧客爽約率
