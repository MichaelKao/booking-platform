## 一、執行摘要

**SOHO Booking Platform** 是一套專為台灣中小型服務業（美容、髮廊、診所、健身房、寵物美容、SPA、餐廳等）打造的 **多租戶 SaaS 預約管理平台**，整合 LINE Bot 智慧客服、CRM 顧客管理、行銷推播、商品銷售、票券系統等功能，讓店家用最低成本數位轉型。

### 核心競爭力

| 優勢 | 說明 |
|------|------|
| **LINE 原生整合** | 台灣 2,100 萬 LINE 用戶，顧客零門檻預約 |
| **多租戶架構** | 一套系統服務無數店家，邊際成本趨近於零 |
| **功能模組化** | 免費基礎 + 付費進階，店家按需訂閱 |
| **AI 智慧客服** | Groq Llama 3.3 免費 API，自動回答常見問題 |
| **全端自研** | 技術自有，無外部依賴，快速迭代 |

### 產品現況（已上線運營）

- **線上網址**：https://booking-platform-production-1e08.up.railway.app
- **技術棧**：Java 17 + Spring Boot 3.2 + PostgreSQL + Redis + Thymeleaf
- **程式碼規模**：34 Controller / 39 Service / 26 Entity / 78+ DTO / 1,206 E2E 測試
- **功能完成度**：18/20 核心功能已完成（見附錄）

### 投資回報預估

**以天使輪 NT$200 萬（佔 18% 股權）為例**：

| 時間點 | 公司估值 | 投資人持股價值 | 投資回報倍數 |
|--------|---------|---------------|-------------|
| 投資時 | NT$1,100 萬 | NT$200 萬 | 1.0x |
| 第 1 年底 | NT$2,000 萬 | NT$360 萬 | **1.8x** |
| 第 2 年底 | NT$5,000 萬 | NT$900 萬 | **4.5x** |
| 第 3 年底 | NT$1.2 億 | NT$2,160 萬 | **10.8x** |

> **估值邏輯**：SaaS 公司估值 = ARR × 10~15 倍（台灣早期通常 8~12 倍）

### 股東分紅政策

| 階段 | 政策 | 說明 |
|------|------|------|
| **虧損期**（Y1） | 不分紅 | 所有收入再投入成長 |
| **損益兩平**（Y2） | 淨利 10% 分紅 | 開始回饋投資人 |
| **獲利期**（Y3+） | 淨利 20~30% 分紅 | 穩定現金流分配 |
| **出場事件** | 依持股比例 | 併購/IPO 時按股權分配 |

**分紅試算（天使輪 18% 股權，第 3 年淨利 NT$508 萬）**：
- 分紅比例：淨利 25% = NT$127 萬
- 投資人分得：NT$127 萬 × 18% = **NT$22.9 萬/年**
- 加上股權增值：總回報 = 分紅 + 持股市值增長

### 退出機制

| 方式 | 時間點 | 預估回報 |
|------|--------|----------|
| **股權回購** | 第 3~5 年 | 原始投資 × 2~5 倍 |
| **策略併購** | 第 3~5 年 | 被 POS/CRM 大廠收購（如 iCHEF、91APP） |
| **下一輪融資** | 第 2~3 年 | A 輪融資時部分退出 |
| **IPO** | 第 5~7 年 | 上創櫃/興櫃 |

## 十二、為什麼投資我們？

### 1. 產品已經做好了

不是 PPT 募資，而是**已上線運營的完整產品**：
- 34 個 API Controller、39 個 Service
- 56 個前端頁面（含 RWD 響應式）
- 1,206 個自動化測試
- LINE Bot 深度整合（7 步驟預約流程）
- AI 智慧客服（Groq Llama 3.3）

### 2. 極低的 Burn Rate

- 目前月營運成本 < NT$200
- 一個人完成市值 NT$200 萬的技術開發
- 募資後大部分資金用於**成長**而非**開發**

### 3. SaaS 模式的複利效應

- 每月訂閱收入自動累積（MRR 只增不減）
- 邊際成本趨近於零（多一個客戶幾乎不增加成本）
- 客戶越多 → 數據越多 → AI 越準 → 產品越好 → 客戶更多

### 4. 巨大的市場機會

- 台灣 12 萬家有預約需求的服務業
- 90% 仍在用紙本或 LINE 群組手動管理
- LINE 整合是殺手級功能（沒有人做得比我們深）

### 5. 清晰的獲利路徑

- 第 14~16 個月損益兩平
- 第 3 年淨利率 50%
- SaaS 估值 = ARR × 10 倍 → 第 3 年估值 **NT$1 億+**
